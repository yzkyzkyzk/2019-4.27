正则

  • 正则(对象):

    • 火星文，难点在于找规律，自己写完之后自己都看不懂

    • 专门用来检索（检查、搜索）模糊范围字符串的一种规则。

  • 简写: //

  • 实例化:new RegExp('字符串','修饰符')

  • 转义符： \

    • **需要注意的是在写 / 记得转义

元字符：\ + 字母

  • 当 \ + 字母 的时候会有特殊含义，这种带有特殊含义的，我们叫元字符

元字符 寓意
\n 换行
\d 一个数字
\D 一个非数字
\s 空格
\S 非空格
\w 一个数字字母下划线
\W 一个非数字字母下划线
\b 一个边界符
\B 一个非边界符

\d+ 多个数字
\D+ 一个非数字

## 修饰符

  • 修饰符:

元字符 寓意
g 全局
i 忽略
m 多行匹配

exec :/规则/.exec(字符串)

  • 语法： /规则/.exec(字符串)

  • 含义：找到正则中第一个匹配的字符且放到数组中。

  • 四个参数[字符、index、input、groups]
    例：

  let str = 'dsd5sa'; 

  console.log(/\d/.exec(str));//["5", index: 3, input: "dsd5sa", groups: undefined]

test :检测正则是否匹配字符串，成立就返回true，否则false

语法：/规则/.test(字符串)

  • 含义：检测正则是否匹配字符串，成立就返回true，否则false
    例：

let str = 'dsd5sa';

console.log(/\d/.test(str));//true

console.log(/b/.test(str));//false

string.match(/正则/) :找到正则中所有匹配的字符且放到数组中。

  • match() 方法可在字符串内检索指定的值，或找到一个或多个正则表达式的匹配。

  • 语法：str.match(/正则/)

    • 找到正则中所有匹配的字符且放到数组中。

    • 没匹配到就是null

  • 量词:

    + 最少1个最多无限
    例： /\d+/多个数字

str.match例子：

let str = 'x1j89321Xeu218nX2d32';

    //全局查找整个字符串，把为数字的存在数组中

    console.log(str.match(/\d/g));

    //["1", "8", "9", "3", "2", "1", "2", "1", "8", "2", "3", "2"]  

    //全局查找整个字符串，把一个数字或者连续多个数字的存在数组中

    console.log(str.match(/\d+/g));

    //["1", "89321", "218", "2", "32"]

    console.log(str.match(/x/ig));

    //["x", "X", "X"]

** replace(要替换的字符串||正则,替换成什么||函数) 替换

  • 语法:
    string.replace(要替换的字符串||正则,替换成什么||函数);

  • 当第二个参数为函数的时候

    • 默认情况:

      • 函数的第一个参数就是每次匹配到的结果

      • 函数的第二个参数index

      • 函数的第三个参数input

      • 函数的第四个（及以后）参数undefined

    • 此函数必须要有return，不然默认为undefined

    • 返回值为新的字符串
      例：

   let str = '王权士姓马';

  console.log(str.replace('王权士','马骅骏'));



    let str = '艹,你在干嘛,练法轮功呢?';   // 艹 靠 尼玛....  &*&#

   console.log(str.replace(/艹|法轮功/g,'*'));//*,你在干嘛,练*呢?

    let s = str.replace(/艹|法轮功/g,function($0,$1,$2,$3){

 console.log($0);//艹|法轮功//要被替换每一项的字符  

 console.log($1);//0|8//被替换的字符出现的索引

 console.log($2)//艹,你在干嘛,练法轮功呢?//整个字符串

 console.log($3)//undefined

        let temp = '';

        for(let i=0;i<$0.length;i++){

            temp += '*';

        }

        return temp;//返回值为新的字符串

    })

    console.log(s);//*,你在干嘛,练***呢?

() 子项、提权

  • () 小括号

  1.()  提权  (1+1)*1

  2.()  子项

  • () 子项： 能够在一个规则中提取某些(指定)字符

  • 注意:
    子项如果包了一个规则，在规则之后有量词，那么结果为最后一个 

 比如:

     let str = '2019'

        let s = str.replace(/(\d)+/,function($0,$1){

            return $1

        })

        console.log(s)//9

            (\d)+  -> 找到的数字不是2是9



 let str = '2019'

        let s = str.replace(/(\d+)/,function($0,$1){

            return $1

        })

        console.log(s)//2019

  • 子项的顺序是从左往右数的，每有一个()就是一个子项,从第二个参数起子项与形参一一对应。
    例：

let str = '2019----4/-/-/-//26'; 

   let s = str.replace(/(\d+)\D+(\d+)\D+(\d+)/,function($0,$1,$2,$3){

        console.log($0);//2019----4/-/-/-//26

        console.log($1);//2019

        console.log($2);//4

        console.log($3);//26    

        return $1 + '年' + $2 + '月' + $3 + '日';

    });

      console.log(s)////2019年4月26日

子项重复项 ： \ + 数字

  • 子项重复项:
    \ + 数字

  • 比如:

  /(d1)c1(b1)\1\2/

  当前有2个子项，一个是d1另一个是b1

  \1：就代表第一个子项，\2：就代表第二个子项

例子：

找出最多的那个数

 let str = '43782635738462353746336372573384633278467382';

 let min = -Infinity;

 let i;

    str = str.split('').sort().join('');

 //22222333333333333344444555666666777777788888

    //比较大小  $0就是当前匹配的 , $1 子项

    str.replace(/(\d)\1+/g,($0,$1)=>{

        console.log($0.length)//5,13,5,6,7,5

        if(min < $0.length){

[ ]: 在中括号中任意选择一个字符

  • []:
    在中括号中任意选择一个字

    • 比如:
      [12345] 可以为1也可以为2也可以为3…
      也可以简写为[1-5],因为 - 是1到5的意思
      为什么能那么写呢？
      因为顺序是按照unicode编码来的
      从大A到小z: [A-z] 不行的; [A-Za-z]

   数字是:[0-9]

   小写英文:[a-z]

   大写英文:[A-Z]

   中文区间范围: [\u4e00-\u9fa5]

   ^: 开头

   $ :结尾

例：

计算下面字符串中的字节长度，假设一个英文字节为1，一个中文为2

[\u4e00-\u9fa5] 中文区间范围 

    let str = 'hello!~张欢,您老贵庚?'; //21个字符

    let num = 0;

    for(let i=0;i<str.length;i++){

        if(/[\u4e00-\u9fa5]/.test(str[i])){

            num += 2;

        }else{

            num ++

        }

    }

   console.log(num);//21

let str = '1088';

    //1088里面有没有108,但是我们的要求是str是否为18-108

    console.log(/1[89]|[2-9][0-9]|10[0-8]/.exec(str));//["108", index: 0, input: "1088", groups: undefined] 没有1088 但是有108就会显示  

    console.log(/^1[89]$|^[2-9][0-9]$|^10[0-8]$/.exec(str));//^: 开头 $ :结尾 

    console.log(/^(1[89]|[2-9][0-9]|10[0-8])$/.exec(str));//从18开始到108结束

    console.log(str.match(/^1[89]$/));

    //只能检测18、19
